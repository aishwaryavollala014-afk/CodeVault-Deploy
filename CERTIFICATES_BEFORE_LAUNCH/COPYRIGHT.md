## Step-by-Step Process
1. Add a copyright notice: `© <year> Gaurav06120714 (CodeVault). All rights reserved.` in footer + `LICENSE`.
2. Keep authorship records (git history establishes provenance).
3. (Optional) Register key works with the copyright office for statutory damages.

## Required Documents
- Authorship records (git log). For registration: the work + application.

## Implementation Guide
- Footer notice in `web-frontend`; header notice in `LICENSE`.
- Ensure contributors assign rights via CLA/DCO so the project holds a clean chain of title.

## Best Practices
- Consistent notices; retain authorship evidence; register before any dispute.

## Common Mistakes
- Assuming a license replaces ownership (it grants rights *from* the owner).
- Accepting contributions without IP assignment.
